# Test cases description

## Test Case 1: USER_AUTHENTICATION_TEST

#### Steps:
1. Open selected webiste - DemoBlaze in my case
2. Sign up with username and password
3. Log in with above credentials
4. Log out

**Expected Result:** User can `sign up`, `log in`, and `log out` successfully.

## Test Case 2: ADD_REMOVE_CART_TEST

#### Steps:
1. Open first product and add it to cart
2. Open cart and remove product

**Expected Result:** Product successfully added to cart and removed after.

## Test Case 3: PRODUCT_PURCHASE_TEST

#### Steps:
1. Open first product and add it to cart
2. Open cart
3. Click `Place Order` button
4. Fill in form fields
5. Click `Purchase` button

**Expected Result:** Purchase successful confirmation appears.
